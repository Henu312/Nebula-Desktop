# 一、产品需求文档（PRD） #

## 项目名称 ##

Nebula Desktop

------

# 1. 项目背景 #

Windows 11 原生桌面存在以下问题：

- 任务栏不可自由移动
- 开始菜单效率低
- 快捷控制中心交互一般
- 缺乏现代化桌面体验
- 对开发者工作流支持较弱

现有方案存在明显缺陷：

| 产品            | 问题                       |
| --------------- | -------------------------- |
| Seelen UI       | 激进，兼容性一般           |
| MyDockFinder    | Bug 多，占用高             |
| Cairo Desktop   | 过重，偏 Shell Replacement |
| Windhawk        | 依赖 Hook，维护复杂        |
| ExplorerPatcher | Windows 更新易失效         |

因此需要一个：

```
轻量
稳定
现代化
兼容微信/QQ
不替代Windows
```

的桌面增强层。

------

# 2. 产品目标 #

开发一个：

```
Windows Desktop Enhancement Layer
```

提供：

- mac 风格任务栏
- 可移动 AppBar
- 现代化快捷控制中心
- 现代化启动器
- 轻量 Dock
- 开发者友好工作流

同时：

```
保留 Windows 原生能力
```

包括：

- Explorer
- Notification Center
- System Tray
- Window Manager

------

# 3. 核心设计原则 #

------

## 3.1 不替代 Windows ##

禁止：

- 替换 explorer.exe
- Shell Replacement
- Hook Explorer 内部实现
- 修改 XAML Tree
- 接管 Notification Center

------

## 3.2 Overlay Architecture ##

项目本质：

```
UI Overlay Layer
```

而不是：

```
Custom Windows Shell
```

------

## 3.3 微信/QQ兼容优先 ##

系统通知：

```
完全保留 Windows 原生通知
```

不读取：

- 消息内容
- 聊天内容

仅显示：

```
哪个应用存在未读/通知
```

例如：

```
微信 ●
QQ ●
企业微信 ●
```

------

## 3.4 低占用优先 ##

目标：

| 指标         | 目标    |
| ------------ | ------- |
| 内存占用     | < 200MB |
| CPU 空闲占用 | < 1%    |
| 启动时间     | < 2s    |

------

# 4. 功能需求 #

------

# 4.1 Taskbar（核心） #

## 功能描述 ##

提供可移动的 mac 风格任务栏。

------

## 支持位置 ##

- Top
- Bottom
- Left
- Right

------

## 功能 ##

### App Launcher ###

固定应用：

- 浏览器
- IDE
- Terminal
- 文件管理器

------

### Running Apps ###

显示：

- 当前运行应用
- 当前激活窗口
- 最小化状态

------

### Notification Badge ###

显示：

```
应用是否有通知
```

不读取通知内容。

------

### Workspace（Phase 2） ###

支持：

- 虚拟工作区
- 多 Workspace

------

## UI风格 ##

支持：

- 毛玻璃
- 圆角
- Hover 动画
- mac 风格布局

------

# 4.2 Dock（可选） #

## 功能 ##

支持：

- 自动隐藏
- Hover 放大
- 固定应用
- 最近应用

------

## 特性 ##

Dock 与 Taskbar：

```
完全独立
```

支持单独关闭。

------

# 4.3 Control Center #

## 功能 ##

mac 风格快捷控制中心。

------

## 第一阶段功能 ##

### 系统控制 ###

- WiFi
- 蓝牙
- 音量
- 亮度
- 夜间模式
- 电源模式

------

## 高级设置 ##

跳转：

```
Windows 原生设置页
```

------

## UI要求 ##

支持：

- 毛玻璃
- 分组卡片
- 动画 Slider
- 深色模式

------

# 4.4 Launcher / Start Menu #

## 风格 ##

推荐：

```
Raycast / Spotlight / PowerToys Run
```

风格。

------

## 功能 ##

### 搜索 ###

支持：

- 应用
- 文件
- 命令
- 最近项目

------

### 固定应用 ###

支持自定义固定。

------

### 最近文件 ###

显示：

- 最近项目
- 最近文档

------

### Command Palette（Phase 2） ###

支持：

```
> open vscode
> ssh server
> git pull
```

------

# 4.5 Notification Indicator #

## 功能 ##

显示：

```
哪个应用存在未读
```

------

## 不支持 ##

禁止：

- 读取聊天内容
- 解析通知内容
- 自定义通知系统

------

## 数据来源 ##

监听：

- Tray Icon 变化
- Window Flash 状态
- Window Attention 状态

------

# 4.6 Widget System（Phase 3） #

支持：

- 天气
- Todo
- CPU
- Docker
- Kubernetes
- Git
- SSH

------

# 5. 非功能需求 #

------

## 稳定性 ##

Windows 更新后：

- 不应导致黑屏
- 不应导致 Explorer 崩溃
- 不应导致无法进入桌面

------

## 兼容性 ##

必须兼容：

- 微信
- QQ
- 企业微信
- 钉钉
- VSCode
- JetBrains IDE

------

## 多屏支持 ##

支持：

- 多显示器
- 不同 DPI
- 不同缩放比例

------

## 深色模式 ##

支持：

- 深色
- 浅色
- 自动切换

------
